# R Markdown Homework Template

This repository provides a simple **R Markdown template for quantitative methods assignments**.
The template helps students organize their homework in a clear and reproducible format that includes:

* Questions
* R code
* Statistical output
* Written explanations

It is designed for courses such as **Quantitative Methods, Political Science Methods, and Data Analysis with R**.

---

# Files

**Homework_Template.Rmd**
Main R Markdown homework template that students can use for assignments.

**Example_Homework_Output.pdf**
Example PDF generated by knitting the template. This shows how the final homework document should look.

**Olympics_HW.dta**
Example dataset used in the template.

# How to Use

1. Download or clone this repository.
2. Open **Homework_Template.Rmd** in RStudio.
3. Make sure the dataset is located in the same folder as the `.Rmd` file.
4. Update the YAML header:

```
title: "Homework X"
author: "Your Name"
date: "`r Sys.Date()`"
```

5. Write your answers under each question.
6. Click **Knit** in RStudio to generate the final PDF.

# Requirements

This template uses R and R Markdown.

Commonly used packages include:

```
tinytex
ggplot2
dplyr
haven
lmtest
stargazer
```

If PDF knitting does not work, install TinyTeX:

```r
tinytex::install_tinytex()
```
